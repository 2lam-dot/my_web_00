GitHub Pages에 프로젝트를 올리는 방법

1. GitHub에 새 저장소(Repository) 생성
   - https://github.com 에 접속 후, 새 저장소를 만듭니다.
   - 저장소 이름은 원하는 대로 설정합니다.

2. 로컬 프로젝트를 저장소에 연결
   - Git이 설치되어 있어야 합니다.
   - 터미널(명령 프롬프트, PowerShell 등)에서 아래 명령어를 실행합니다.

   ```powershell
   cd "c:\Users\enfka\Desktop\coding\my_web_00"
   git init
   git remote add origin https://github.com/사용자명/저장소명.git
   git add .
   git commit -m "first commit"
   git push -u origin master
   ```
   - 사용자명과 저장소명은 본인 계정/저장소에 맞게 변경하세요.

3. GitHub Pages 설정
   - GitHub 저장소 페이지로 이동합니다.
   - Settings > Pages 메뉴로 이동합니다.
   - Branch를 `master` 또는 `main`으로 선택하고 `/ (root)`를 선택합니다.
   - 저장하면 배포 주소가 생성됩니다.

4. 배포 주소 확인
   - 잠시 후, 안내된 URL에서 웹사이트를 확인할 수 있습니다.

추가 팁:
- `index.html` 파일이 루트에 있어야 정상적으로 동작합니다.
- 변경사항이 있을 때마다 `git add .`, `git commit -m "메시지"`, `git push`로 업데이트하세요.

끝.
